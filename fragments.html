ButterflyDreaming : PROJECT DESIGN

BITS AND PIECES OF THE FUTURE SYSTEM

On this page you can experiment with the idea of text controlling media. You can change text, use a more complicated interface to change the text, copy and paste text to store it. Or copy a link that will send a text description together with an interpreting web site as a single link. Just play around and hopefully things will become clear. In the finished platform they would probably live in a little panel and consume text back and forwards from the editing of a node in the graph. Or they can live outside the system as independent players reading text from ButterflyDreaming and writing back to other destinations. There are many possibilities. (Developers can access the code freely at GitHub and create their own modules and extension directives.)
